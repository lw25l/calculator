# 智慧景区 AI 数字人导游（微信小程序示例）

本仓库新增了一个可直接导入微信开发者工具的 `miniprogram` 目录，实现赛题要求的核心能力：

- 游客端：多模态交互入口（文本已落地，语音接入点预留）、智能问答、个性化路线推荐。
- 管理端：知识库管理、数字人配置展示、游客感受度分析、运营数据概览。
- 技术路径：本地知识库 mock + 可扩展到 RAG、多模态大模型 API、ASR/TTS 与数字人驱动引擎。

## 目录结构

- `miniprogram/app.js`：全局配置（游客兴趣、音色、数字人形象）。
- `miniprogram/app.json`：页面路由与窗口配置。
- `miniprogram/pages/index`：游客首页与个性化路线推荐。
- `miniprogram/pages/chat`：智能问答对话页。
- `miniprogram/pages/admin`：后台管理与情感分析页。
- `miniprogram/utils/mockData.js`：示例知识库、路线推荐、情感分析逻辑。

## 使用方式

1. 打开微信开发者工具。
2. 选择“导入项目”，项目目录选择仓库下的 `miniprogram`。
3. 运行后可在首页进入游客交互与管理后台页面。

## 赛题能力对齐说明

- 多模态交互：当前支持文本问答；已预留语音ASR/TTS和数字人口型联动接入位置。
- 智能问答：通过本地知识库示例完成事实问答流程。
- 个性化推荐：按“历史文化/自然风光/亲子休闲”兴趣输出不同游览路线。
- 管理后台：支持知识库维护、反馈情感评分与核心运营指标看板展示。
